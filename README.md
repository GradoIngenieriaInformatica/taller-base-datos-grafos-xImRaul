### Taller

* Cada alumno tiene un archivo con preguntas seleccionadas al zar en base a un banco d epreguntas y una plantilla de estructura.
* El repo tiene un archivo llamado .examen_asignado.json, éste no debe ser modificado, está protegido por hash y cualquier cambio será detectado durante ala evaluación automática.
* Resolver los enunciados en los archivos dentro del directorio respuestas, escribir  únicamente la respuesta solicitada, sin repetir nunciado, ni poner ningún tipo de texto introductorio.
* La evaluación masiva automática se realizará después que todos hayan terminado el taller fuera dle horario de clases.
